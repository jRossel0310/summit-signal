# 3D Terrain View Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a "3D terrain" toggle to the Layers panel that drapes the map over the existing DEM and tilts the camera, so terrain rises in real relief.

**Architecture:** A new registry overlay `overlay.terrain3d` (kind `terrain-3d`) renders as a Terrain-group checkbox via the existing layer plumbing; MapView special-cases it in a `sync3DTerrain()` that calls `setTerrain`/sky/`easeTo` on the DEM it already loads, re-applied via `syncAll()` so it survives basemap style swaps.

**Tech Stack:** Vite + React + TypeScript, MapLibre GL, vitest. No new dependencies, no backend changes.

## Global Constraints

- No new dependencies.
- Fixed vertical exaggeration ~1.4×; no slider.
- Off by default each session; not persisted.
- Reuse the existing `dem` raster-dem source; do not add a new DEM source.
- The new overlay must NOT be added to MapView's `OVERLAY_RENDER` (it is not a draw layer).
- `overlay.terrain3d` must have a non-empty `description` (the registry test enforces it).
- Sky is best-effort and guarded (`typeof map.setSky === "function"`); terrain works without it.

- [ ] **Step 2: Add the sky helper and `sync3DTerrain()`**

Find the `syncVisibility` function (it sets layer visibility/opacity from `OVERLAY_RENDER`). Immediately AFTER that function's closing brace, add:

```typescript
  function applySky(map: maplibregl.Map, on: boolean) {
    const m = map as unknown as { setSky?: (s?: unknown) => void };
    if (typeof m.setSky !== "function") return;   // older MapLibre: skip sky, terrain still works
    m.setSky(
      on
        ? {
            "sky-color": "#9ec4e8",
            "sky-horizon-blend": 0.5,
            "horizon-color": "#dfeaf4",
            "horizon-fog-blend": 0.5,
            "fog-color": "#f4f2ec",
            "fog-ground-blend": 0.5,
          }
        : undefined,
    );
  }

  function sync3DTerrain() {
    const map = mapRef.current;
    if (!map) return;
    const on = !!layerState["overlay.terrain3d"]?.visible;
    if (on) {
      if (map.getSource("dem")) {
        map.setTerrain({ source: "dem", exaggeration: TERRAIN_EXAGGERATION });
      }
      applySky(map, true);
      if (map.getPitch() < 1) map.easeTo({ pitch: TERRAIN_PITCH, duration: 600 });
    } else {
      map.setTerrain(null);
      applySky(map, false);
      if (map.getPitch() > 1) map.easeTo({ pitch: 0, duration: 600 });
    }
  }
```
